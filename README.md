A simple, secure password manager for the CLI! Written in Python it allows storing and managing credentials locally in an encrypted database.

## Features
- Encrypted vault
- Password generator (customizable length & character sets)

## Quickstart

1. Install `cryptography` module:

```bash
pip install cryptography
```

2. Run the `SecurePasswordManager.py` file:


```bash
py SecurePasswordManager.py
```
